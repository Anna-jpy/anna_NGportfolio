Déploiement complet d'une solution GLPI (Gestionnaire Libre de Parc Informatique) sur serveur Debian 12.5. Installation et configuration de l'infrastructure serveur web complète avec Apache2, PHP et MariaDB. Mise en place de phpMyAdmin pour la gestion de la base de données et sécurisation de la plateforme.

  • Installation et configuration de Debian 12.5 (Bookworm)
  • Mise en place de l'environnement LAMP (Apache2, PHP 8.2, MariaDB)
  • Installation de GLPI version 10.0.14
  • Configuration et sécurisation de la base de données MariaDB
  • Installation de phpMyAdmin pour la gestion de la base de données
  • Sécurisation du système (suppression install.php, configuration PHP sécurisée)
  • Configuration des permissions et des dossiers de données
  • Mise en place de l'interface web et configuration initiale

Dev Girls - Plateforme de Petites Annonces (Leboncoin)

Équipe : Annaelle MPASSI, Gabrielle KEMGUAN, Merveille | Rôle : Développement full-stack

Développement d'une plateforme web de petites annonces inspirée de Leboncoin, permettant aux utilisateurs de publier, consulter et gérer des annonces de téléphones. Application complète avec authentification sécurisée, système de messagerie, gestion de compte utilisateur et fonctionnalités CRUD complètes.

  • Système d'inscription et de connexion sécurisé avec hashage des mots de passe
  • Gestion complète des annonces (création, modification, suppression, affichage)
  • Système de filtrage avancé (par marque, prix min/max)
  • Recherche textuelle dans les annonces
  • Système de messagerie entre utilisateurs
  • Upload et gestion d'images pour les annonces
  • Gestion de profil utilisateur et mon compte
  • Interface responsive avec design moderne
