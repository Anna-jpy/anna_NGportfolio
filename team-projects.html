Installation et configuration GLPI

Installation et Configuration GLPI

Équipe : 4 étudiants (Annaelle Mpassi, Rahis Madimean, Merveil Mamno, Perine Cheteu)
Durée : 96 heures (4 jours)
Rôle : Administrateur système et base de données

Déploiement complet d'une solution GLPI sur serveur Debian 12.5. Installation et configuration d'une infrastructure serveur web avec Apache2, PHP et MariaDB, mise en place de phpMyAdmin pour la gestion de la base de données et sécurisation globale de la plateforme.

  • Installation et configuration de Debian 12.5 (Bookworm)
  • Mise en place de l’environnement LAMP : Apache2, PHP 8.2, MariaDB
  • Installation de GLPI version 10.0.14
  • Configuration et sécurisation de la base de données MariaDB
  • Installation de phpMyAdmin pour la gestion de la base de données
  • Sécurisation du système avec suppression de install.php et réglages PHP
  • Configuration des permissions et des dossiers de données
  • Mise en place de l’interface web et configuration initiale
Technologies utilisées
GLPI Debian 12 Apache2 PHP 8.2 MariaDB phpMyAdmin Linux

Projet web
Plateforme de petites annonces

Dev Girls - Plateforme de Petites Annonces (Leboncoin)

Équipe : Annaelle MPASSI, Gabrielle KEMGUAN, Merveille
Rôle : Développement full-stack

Développement d'une plateforme web de petites annonces inspirée de Leboncoin, permettant aux utilisateurs de publier, consulter et gérer des annonces de téléphones. Le projet intègre une authentification sécurisée, une messagerie entre utilisateurs, la gestion des profils et les opérations CRUD complètes.

  • Système d’inscription et de connexion sécurisé avec hashage des mots de passe
  • Gestion complète des annonces : création, modification, suppression et affichage
  • Système de filtrage avancé par marque et par prix
  • Recherche textuelle dans les annonces
  • Système de messagerie entre utilisateurs
  • Upload et gestion d’images pour les annonces
  • Gestion de profil utilisateur et espace “Mon compte”
  • Interface responsive avec design moderne
Technologies utilisées
PHP MySQL HTML/CSS JavaScript Sessions Upload de fichiers
